---
title: Widget Library
---

1. Data Source: 
   Fitur ini digunakan untuk mengimpor data dari berbagai sumber seperti CSV, Excel, dan database. Pengguna dapat dengan mudah mengakses data lokal maupun data dari sumber eksternal melalui widget File, widget CSV Import, widget Dataset, widget SQL Table, atau plugin khusus yang mendukung koneksi ke sumber data lain.
2. Data Extraction: 
   Fitur ini memungkinkan pengguna dalam proses pengambilan data dari dataset yang lebih besar untuk dianalisis lebih lanjut. Dengan widget seperti Data Sampler, Select Columns, Select Row,Â  dan Transpose, pengguna dapat mengekstrak dan menyaring bagian data yang diinginkan. Hal ini mempermudah pengambilan hanya variabel atau subset data tertentu untuk memfokuskan analisis.
3. Data Quality: 
   Fitur ini memungkinkan pengguna untuk meningkatkan kualitas data, seperti mendeteksi data yang hilang (missing values) dan menangani data yang tidak konsisten. Widget seperti Randomize, Discretize, Impute digunakan untuk mengisi nilai yang hilang, sedangkan Preprocess membantu dalam normalisasi dan transformasi data. Hal ini memastikan data yang digunakan dalam analisis memiliki standar yang layak untuk menghasilkan model yang andal.
4. Data Discovery/Modeling: 
   Fitur ini memungkinkan pengguna dalam proses penemuan data (data discovery) dan pemodelan dengan menyediakan workflow untuk proses eksplorasi data. Pengguna dapat menggunakan widget seperti Linear Regression, Logistic Regression, Naive Bayes, Decision Tree, Random Forest, K-Nearest Neighbor (KNN), K-Means, dan Support Vector Machine (SVM) untuk memudahkan pengguna membangun model prediktif dan analisis klaster tanpa menulis satu baris kode pun.
5. Data Visualization: 
   Fitur ini memungkinkan pengguna untuk mempresentasikan data dalam berbagai bentuk grafik interaktif. Dengan widget seperti Histogram, Scatter Plot, Mosaic Display,Bar Plot, Heat Map, Venn Diagram, dan Line Plot, pengguna dapat membuat visualisasi untuk mempermudah pemahaman terhadap distribusi data, korelasi antar variabel, serta hasil analisis dan pemodelan. Hal ini membantu dalam membuat keputusan berbasis data yang lebih cepat dan tepat.
