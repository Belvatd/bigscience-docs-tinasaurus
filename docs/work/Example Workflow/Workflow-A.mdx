### Goals : 

Menampilkan informasi banyaknya orang yang membayar pajak berdasarkan tanggal/waktu pembayaran pajak.

![](</img/Workflow/Workflow-1/EX1-1.png>)

Step : 

1.Anda dapat mengunggah sumber data menggunakan widget File. Klik widget File >Input File > Set Columns > Klik tombol Apply

 ![](</img/Workflow/Workflow-1/EX1-2.png>)

 Result ![](</img/Workflow/Workflow-1/EX1-3.png>)

2\. Gunakan Widget Impute untuk menghilangkan instance yang kosong. Klik widget Impute > pilih Remove instances with unknown values pada Default Method. Berdasar file yang diinput / raw data terdapat 375 instance. Setelah di impute terdapat 53 instance yang nilai nya tidak kosong. 

![](</img/Workflow/Workflow-1/EX1-4.png>)

 

Result![](</img/Workflow/Workflow-1/EX1-5.png>)


3\. Gunakan widget Distribution untuk memvisualisasikan informasi banyaknya orang yang membayar pajak berdasarkan tanggal atau waktu pembayaran pajak. Klik widget Distribution > pilih kolom tanggal atau waktu pembayaran sebagai atribut utama untuk visualisasi. Kemudian atur tampilan histogram untuk menampilkan distribusi data yang jelas.

![](</img/Workflow/Workflow-1/EX1-6.png>)

#### Hasil Analisis dari Diagram Histogram

* Tanggal dengan pembayaran pajak terbanyak.
* Tren pembayaran pajak, seperti apakah pembayaran lebih banyak dilakukan di akhir bulan, di awal minggu, atau pada hari tertentu. Informasi ini dapat membantu dalam memahami kebiasaan wajib pajak.

#### Insight

* Mengidentifikasi waktu terbaik untuk mengingatkan wajib pajak tentang pembayaran pajak.
* Menyusun strategi untuk meningkatkan kepatuhan pajak dengan cara mengoptimalkan pengumuman atau kampanye pada waktu-waktu tertentu berdasarkan pola pembayaran.
