title: Windows
---

1. Unduh aplikasi pada [BigScience-1.0.0-Phyton311-win\_amd64.exe](https://drive.google.com/file/d/10meq9JbZs48QQpP2xH3jWBMR_aO7IsBg/view?usp=drive_link)
2. Jalankan installer dengan cara *double-click*. Muncul kotak dialog dan klik ***Next*** untuk memulai instalasi aplikasi Big Science.\
   ![](/img/instalasi-windows/1.png)
3. Tampil halaman License Agreement, klik tombol ***I Agree*** untuk melanjutkan tahapan instalasi.![](/img/instalasi-windows/2.png)
4. Pilih ***Install*** ***just for me*** untuk jenis pengguna aplikasi dan klik ***Next.***![](/img/instalasi-windows/3.png)
5. *Checklist* semua  komponen yang akan di install, dan klik ***Next.***![](/img/instalasi-windows/4.png)
6. Pilih lokasi folder aplikasi yang akan di instal dengan tombol\*\*\* Browse\*\*\*, dan klik ***Next*** untuk melanjutkan tahapan instalasi. ![](/img/instalasi-windows/5.png)
7. Klik ***Install.***![](/img/instalasi-windows/6.png)
8. Muncul *pop-up* konfirmasi untuk instalasi library Python. Klik ***OK*** untuk mengunduh library.
9. Instal library Python dengan cara ***checklist Add python.exe to Path*** dan Pilih ***Install Now.***![](/img/instalasi-windows/10.png)
10. Proses Instalasi library Python sedang berlangsung. Tunggu beberapa saat hingga selesai. Klik tombol ***Close*** untuk melanjutkan instalasi aplikasi Big Science.![](/img/instalasi-windows/11.png)
11. Tunggu beberapa saat hingga proses Instalasi aplikasi  Big Science telah selesai, lalu  klik ***Next.***![](/img/instalasi-windows/8.png)
12. Klik ***Finish*** untuk mengakhiri proses instalasi aplikasi.![](/img/instalasi-windows/9.png)
