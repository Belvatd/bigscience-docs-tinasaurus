1. Unduh aplikasi pada [BigScience-1.0.0-Phyton311-win\_amd64.exe](https://drive.google.com/file/d/10meq9JbZs48QQpP2xH3jWBMR_aO7IsBg/view?usp=drive_link)
2. Jalankan installer dengan cara *double-click*. Muncul kotak dialog dan ikuti perintah instalasi dengan cara *drag and drop* aplikasi Big Science ke folder **Application.**\
   \
   ![](/img/instalasi-windows/12.png)
3. Tunggu beberapa saat hingga proses instalasi selesai.\
   \
   ![](/img/instalasi-windows/13.png)
4. Aplikasi telah terinstal dan siap digunakan.\
   ![](/img/instalasi-windows/14.png)
