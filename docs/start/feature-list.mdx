---
title: Feature List
---

* **Widget Customization with drag & drop functionality (low-code app)**\
  BigScience menawarkan antarmuka berbasis drag-and-drop untuk membangun workflow analisis data tanpa perlu menulis banyak kode (low-code development). Pengguna dapat dengan mudah melakukan drag-and-drop widget ke canvas dan menyesuaikan widget yang sudah ada atau menambahkan fungsi baru dengan integrasi Python. Dengan BigScience, pengguna dengan mudah melakukan eksplorasi dan analisis data tanpa memerlukan penulisan kode atau query.

 

* **Visual Data Workflow**\
  Worklfow data di BigScience dirancang secara visual yang memungkinkan pengguna untuk merancang proses analisis data secara modular. Berikut widget yang dapat mendukung pembuatan workflow Anda :
  * Data Source
    BigScience menyediakan widget yang dapat digunakan untuk koneksi data dari aplikasi internal, database maupun OTA / media sosial. Widget tersebut berupa widget File, Dataset, SQL Table.
  * Data Extraction
    BigScience menyediakan widget untuk melakukan proses ekstraksi data. Widget tersebut berupa widget Sampler, Select Column, Select Row, Transpose
  * Data Quality
    BigScience menyediakan widget untuk melakukan  proses meningkatkan kualitas data yang mengacu pada tingkat keakuratan, kelengkapan dan relevansi data. Widget tersebut berupa Discretize, Randomize, Preprocess, Purge/Apply Domain
  * Data Discovery/Modeling
    BigScience menyediakan widget untuk melakukan  proses eksplorasi data untuk mengidentifikasi pola, korelasi dan distribusi data serta membuat model machine learning. Widget tersebut berupa Linear Regression, Logistic Regression, Naive Bayes, Decision Tree, Random Forest, K-Nearest Neighbor (KNN), K-Means, dan Support Vector Machine (SVM).
  * Data Visualization
    BigScience menyediakan widget untuk melakukan  proses visualisasi data dalam bentuk visual yang informatif dan mudah dipahami. Widget tersebut berupa Venn Diagram, Line Plot, Bar Plot, Scatter Plot, dan Heat Map

 

* **Add On & Plugin**\
  BigScience mendukung penambahan add-on seperti Text Mining yang dirancang khusus untuk analisis data dan plugin untuk memperluas fungsi aplikasi  sesuai kebutuhan pengguna.
* **Data Modeling**\
  BigScience menyediakan fitur pembelajaran mesin (machine learning) untuk membangun dan mengevaluasi model secara visual.
