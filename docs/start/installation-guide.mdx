---
title: Installation Guide
---

1. Unduh aplikasi pada [BigScience-1.0.0-Phyton311-win\_amd64.exe](https://drive.google.com/file/d/10meq9JbZs48QQpP2xH3jWBMR_aO7IsBg/view?usp=drive_link)
2. Jalankan installer dengan cara double-click. Muncul kotak dialog dan klik Next untuk memulai instalasi aplikasi Big Science. \
   ![](/img/conflu.png)
3. Tampil halaman License Agreement, klik tombol I Agree untuk melanjutkan tahapan instalasi. 
4. Pilih Install just for me untuk jenis pengguna aplikasi dan klik  Next.
