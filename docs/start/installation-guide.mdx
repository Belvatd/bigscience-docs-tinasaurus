---
title: Installation Guide
---

1. Unduh aplikasi pada [BigScience-1.0.0-Phyton311-win\_amd64.exe](https://drive.google.com/file/d/10meq9JbZs48QQpP2xH3jWBMR_aO7IsBg/view?usp=drive_link)
2. Jalankan installer dengan cara *double-click*. Muncul kotak dialog dan klik ***Next*** untuk memulai instalasi aplikasi Big Science. 
3. Tampil halaman License Agreement, klik tombol ***I Agree*** untuk melanjutkan tahapan instalasi.
4. Pilih Install ***just for me*** untuk jenis pengguna aplikasi dan klik ***Next.***
5. Checklist semua  komponen yang akan di install, dan klik Next. 
6. Pilih lokasi folder aplikasi yang akan di instal dengan tombol Browse, dan klik Next untuk melanjutkan tahapan instalasi
7. Klik Install. 
8. Muncul pop-up konfirmasi untuk instalasi library Python. Klik OK untuk mengunduh library. 
9. Instal library Python dengan cara checklist Add python.exe to Path dan Pilih Install Now. 
10. Proses Instalasi library Python sedang berlangsung. Tunggu beberapa saat hingga selesai. Klik tombol Close untuk melanjutkan instalasi aplikasi Big Science.
11. Tunggu beberapa saat hingga proses Instalasi aplikasi  Big Science telah selesai, lalu  klik Next. 
12. Klik Finish untuk mengakhiri proses instalasi aplikasi.
