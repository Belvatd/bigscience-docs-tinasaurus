---
title: App Requirements
---

Aplikasi BigScience adalah platform analisis data yang menyediakan berbagai alat untuk pemrosesan, analisis, dan visualisasi data. Untuk memastikan aplikasi berjalan dengan optimal, berikut adalah persyaratan sistem dan perangkat lunak yang harus dipenuhi.

| Aspek              | Minimal Spesifikasi            |
| ------------------ | ------------------------------ |
| Operating System   | Windows 10 atau lebih (64-bit) |
| RAM                | Min 4GB atau lebih (8GB)       |
| Local Storage      | 500 MB free space              |
| Python Environment | Python versi 3.11.8            |

Jika Anda mengalami kesulitan saat menginstal atau menggunakan Orange, silakan kunjungi halaman Support.
